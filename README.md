# react-native-base64-image-converter

library

## Installation

```bash
yarn add react-native-base64-image-converter
cd ios && pod install
```

## Usage

```js
import { imageToBase64 } from 'react-native-base64-image-converter';

// ...

const result = imageToBase64(`file://imagepath`);
```
